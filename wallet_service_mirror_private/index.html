The private side of wallet-service-mirror. This program forms outgoing connections to both a wallet service instance, as well as a public wallet-service-mirror instance. It then proceeds to poll the public side of the mirror for requests which it then forwards to the wallet service. When a response is received it is then forwarded back to the mirror.
