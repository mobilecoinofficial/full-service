# Sync Txos

## Request

| Param           | Requirements                                              |
|-----------------|-----------------------------------------------------------|
| `mnemonic`      | Must be a valid 24 word mnemonic                          |
| `txos_unsynced` | list of unsynced txos from the VO account in full service |
