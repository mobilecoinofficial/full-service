# Remove Gift Code

## Parameters

| Required Param  | Purpose                                | Requirements                           |
|-----------------|----------------------------------------|----------------------------------------|
| `gift_code_b58` | The base58-encoded gift code contents. | Must be a valid b58-encoded gift code. |
