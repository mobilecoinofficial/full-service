# Update Account Name

## Parameters

| Required Param | Purpose                                      | Requirements                      |
|----------------|----------------------------------------------|-----------------------------------|
| `account_id`   | The account on which to perform this action. | Account must exist in the wallet. |
| `name`         | The new name for this account.               |                                   |
